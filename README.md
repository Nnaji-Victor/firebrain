# ✨ How To Use
1. **Install and switch to the correct version of Node using NVM**

   ```sh
   nvm install
   ```
2. **Install dependencies**

   ```sh
   yarn install
   ```
3. **Start the development server**

   ```sh
   yarn start
   ```
